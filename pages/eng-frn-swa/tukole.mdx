# Tukole

Muuyegerere tukolanwe, tuzamuule indeto yitu Kifuliiru.

Tukole is a Kifuliiru word that means let's work.
This is an invitation to whoever wants to help this project.

Mugweeti ibiindu muloziizi ukubuuza kandi li muloziizi ukutabaala? [Tuyandikire hano](https://ibufuliiru.editorx.io/ibufuliiru/tuyandikire).
